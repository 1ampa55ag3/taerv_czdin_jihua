- 文件组织方案

  - 按词组音节（忽略声调）分配文件，同音词(忽略声调)组织在同一文件中。

  - 每个词条按照以下方式编写：

    ```md
    # 中文
    zhon1 ven2
    - 意义一
    	- 例子一
    	- 例子二
    - 意义二
    	- 例子三
    ```

  - 词条之间不留空行
